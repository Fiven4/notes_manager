# 📌 О проекте

**Notes Manager** — это консольное приложение на Python, реализующее систему управления заметками пользователей.

Проект демонстрирует практическое использование:

- работы с файлами
- многопоточности (`threading`)
- логирования действий
- системы пользователей (регистрация и авторизация)
- хранения данных в структурах Python

Каждый пользователь может создавать и управлять собственными заметками, а система автоматически сохраняет данные через отдельный поток.

Проект создан как учебное приложение для демонстрации архитектуры простых CLI-программ.

---

# 🚀 Основные возможности

### 👤 Управление пользователями
- регистрация пользователей
- авторизация в системе
- хранение учетных данных

### 📝 Управление заметками
- создание заметок
- просмотр заметок
- удаление заметок
- привязка заметок к конкретному пользователю

### 🔄 Автоматическое сохранение
- отдельный поток автоматически сохраняет заметки
- сохранение происходит каждые **10 секунд**

### 📜 Логирование
Все действия пользователей записываются в файл:

```
actions.log
```

Примеры логов:

```
[INFO] [12.05.2025 14:21:30] [admin] – Добавлена заметка
[WARNING] [12.05.2025 14:22:10] [user] – Неудачная попытка входа
[ERROR] [12.05.2025 14:22:50] [admin] – Ошибка автосохранения
```

# 🏗 Архитектура проекта

Программа построена вокруг нескольких ключевых компонентов:

### 1️⃣ Система пользователей

Пользователи хранятся в словаре:

```python
users = {
    "username": "password"
}
```

Каждый пользователь имеет собственный список заметок.

---

### 2️⃣ Хранение заметок

Заметки организованы следующим образом:

```python
notes = {
    "username": [
        "Первая заметка",
        "Вторая заметка"
    ]
}
```

Это позволяет легко связывать данные с конкретным пользователем.

---

### 3️⃣ Многопоточность

Для автосохранения используется **отдельный поток**:

```python
threading.Thread(target=auto_save)
```

Он работает параллельно с основной программой и:

- периодически сохраняет заметки
- не блокирует работу пользователя

---

### 4️⃣ Логирование

Все действия записываются функцией:

```python
log_action()
```

Каждая запись содержит:

- тип события
- дату и время
- пользователя
- описание действия

# 💾 Автосохранение

Программа автоматически сохраняет заметки в файл:

```
Desktop/notes.txt
```

Сохранение происходит **каждые 10 секунд**.

Пример содержимого файла:

```
user1:
  Купить продукты
  Сделать домашнее задание

admin:
  Подготовить презентацию
```
